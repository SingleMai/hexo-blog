利用透明度实现悬停提示效果

也是看别人用到了，才发现这里有兼容性问题。参考至CSS - firefox与IE透明度(opacity)设置区别

IE：

filter:alpha(opacity=sqlN)
其中 sqlN的值域为[0, 100]

js: ieNode.style.filter="alpha(opacity=sqlN)";

Firefox：/参考,不推荐使用/

-moz-opacity:sqlN
其中sqlN的值域为[0, 1]

Firefox,Chrome和Safari：

opacity:sqlN
其中sqlN的值域为[0, 1]

js:firefoxNode.style.opacity=sqlN;

参考至CSS - firefox与IE透明度(opacity)设置区别
